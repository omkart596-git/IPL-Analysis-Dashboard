# IPL Analysis Dashboard – Power BI Project
**Project Overview**
This Power BI dashboard analyzes IPL season data to provide insights into:
Total Runs & Wickets |
Orange Cap & Purple Cap Winners |
Team Performance Analysis |
Season-wise Comparisons |
Player Statistics |
Super Over Matches Analysis

**Tools Used**
Power BI |
Data Cleaning & Transformation |
DAX Measures |
Data Modeling

**Key Insights**
Identified top-performing batsmen and bowlers season-wise |
Analyzed team performance trends |
Compared seasonal statistics using dynamic filters
